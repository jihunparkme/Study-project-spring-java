## 기능 요구사항

- [x] car.Sonata 2대, car.Avante 1대, car.K5 2대로 총 5대의 차량을 보유
- [x] 고객이 인터넷으로부터 예약할 때 여행할 목적지의 대략적인 이동거리를 입력
- [x] 이 이동거리를 활용해 차량 별로 필요한 연료를 주입
- [x] 차량 별로 주입해야 할 연료량을 확인할 수 있는 보고서를 생성
- [x] 각 차량별 연비
    ```
    * car.Sonata : 10km/리터
    * car.Avante : 15km/리터
    * car.K5 : 13km/리터
    ```
  
## 프로그래밍 요구사항

- [x] 상속과 추상 메소드를 활용
- [x] 위 요구사항을 if/else 절을 쓰지 않고 구현
- [ ] 인터페이스를 적용한 구현
